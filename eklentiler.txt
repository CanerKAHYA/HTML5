1-/LiveServer [Canlı Sunucu]/:Canlı bir ön izleme sunarak HTML dosyalarınızı anında tarayıcıda görmenizi sağlar.

2-/Emmet /:Hızlı ve etkili HTML ve CSS kodu yazmanıza yardımzcı olan güçlü bir eklenti.
//*Örneğin*\:"html:5"yazarak bir HTML5  şablonu oluşturabilirsiniz.\

3-/HTML Boilerplate [HTML Basmakalıp]/:HTML dosyalarınızı başlatmak için temel bir şablona sahip olmanızı sağlar.

4-/Auto Rename Tag[Otomatik Etiket Yeniden Adlandırma]/:Bir HTML etiketini yeniden adlandırdığınızda eşleşen kapanış etiketini otomatik olarak günceller.

5-/HTML CSS Support[HTML CSS Desteği]/:HTML içinde CSS sınıf ve ID önerileri sunar ve CSS dosyalarınıza kolayca bağlantı eklemenizi sağlar.

6-/Html Snippets[Html Parçacıkları]/:Hızlıca HTML kodlarınızı oluşturmanız için kullanışlı kısa yollar sunar.

7-/HTMLHint/:HTML kodunuzu denetler ve hataları veya stil kılavuzu ihlallerini gösterir.

8-/HTML to CSS autocompletion[HTML'den CSS'ye otomatik tamamlama]/:CSS dosyalarında HTML etiketlerine otomatik tamamlama önerileri sunar.

9-/Auto Close Tag[Otomatik Kapatma Etiketi]/: -Bir HTML etiketi yazdığınızda otomatik olarak kapanış etiketini ekler.

10-/Prettier/:HTML ve diğer kodları otomatik olarak biçimlendirir ve düzenler
